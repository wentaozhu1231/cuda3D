Glirt:GPU-Accelerated Linear Image Registration Toolkit, 是我的开源计划 GMTK(Accelerated Medical Toolkit，包含Medical Imaging, Medical Image Registration and Segmentation, etc)的第一部分. 
整个开源项目将在gmtk.org上公布，在我完成glirt的UI后，将会发布第一个beta版，同时开放gmtk.org. 
医学图像的线性配准比非线性配准更不好加速，使用了很多的技巧才完成了这个代码，详细的请查看下载包中的论文。虽然加速比只有20倍左右，但是这已经是大大优化之后的结果了。
奉献给大家，希望对大家学习CUDA，以及医学图像计算的能有帮助。
